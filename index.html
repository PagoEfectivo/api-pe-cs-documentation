    PagoEfectivo API

    Objetivo

    PagoEfectivo API fue creado para que el afiliado integre en su plataforma las funcionalidades de generación, consulta, eliminación y actualización de los CIPS. Se ha desarrollado una Demo de ejemplo para mostrar cómo consumen los métodos en el sistema operativo windows, usando el lenguaje de programación C#.

    Al descargar el código fuente y ejecutar la Demo se observa que contiene un menú con los métodos y una opción llamada Configuración donde se debe introducir los datos de accesos facilitados por comercial.

    Requisitos

    • Microsoft Visual Studio 2017 u otro Editor.
    • .NET Core 2.0.

    Métodos

    A continuación se muestra con detalle la invocación de los siguientes métodos:

    Método de Autorización

    Se encarga de devolver la información de los cips solicitados. Para consumir el método generar se requieren los siguientes parámetros:

    • URL: http://services.pagoefectivo.pe

    • Punto de entrada: /v1/authorizations

    • Método: POST

    Al ejecutar la Demo es necesario registrar el id de servicio, Access key, secret key para hacer uso de las funcionalidades como se muestra a continuación:

    Luego de introducir los datos de autorización, clic en registrar, luego muestra la confirmación.

    Cada vez que se consumen los métodos de PagoEfectivo API es necesario generar los accesos de seguridad enviando los datos ingresados del formulario Configuración. La demo guarda los datos enviados en memoria Caché para Agilizar la carga y enviarlos al servicio de autorización.

    A continuación se muestra el código desarrollado en C# para consumir el servicio de autorización.

    //Método Autorización CIP
...
Authenticate authorization = new Authenticate()
{
    accessKey = authenticateViewModel.AccessKey,
    idService = authenticateViewModel.IdService,
    dateRequest = DateNow.ToString("yyyy-MM-ddTHH:mm:sszzz")
};

var authRequest = authorization.idService.ToString() + "." + authorization.accessKey + "." + authenticateViewModel.SecretKey + "." + authorization.dateRequest;

authorization.hashString = Hash.HashString(authRequest);

using (var client = new HttpClient())
{
    client.BaseAddress = new Uri(_url);
    string stringData = JsonConvert.SerializeObject(authorization);
    var contentData = new StringContent(stringData, System.Text.Encoding.UTF8, "application/json");
    HttpResponseMessage response = client.PostAsync("v1/authorizations", contentData).Result;

    if (response.IsSuccessStatusCode)
    {
        dynamic responseAuth = JsonConvert.DeserializeObject<AuthenticateResponse>(response.Content.ReadAsStringAsync().Result);
        responseAuth.url = _url;
        return responseAuth;
    }
    else
        return response;
}

    Método Generar CIP

    Se encarga de devolver la información de los cips solicitados. Para consumir el método generar se requieren los siguientes parámetros:

    • URL: http://services.pagoefectivo.pe

    • Punto de entrada: /v1/cips

    • Método: POST

    Se requiere ingresar el tipo de moneda, monto, código de transacción y correo de usuario para la generación de CIP. El formulario tiene la opción de habilitar campos opcionales para Generar un CIP con un mayor detalle.

    Response: Al Ingresar los Datos solicitados clic en el botón generar. se muestra la siguiente respuesta:

    Es posible registrar datos opcionales para generar cip con mayor detalle de información, como se muestra a continuación.

    Clic en botón generar y muestra la respuesta del servicio.

    Para visualizar el Cip generado hacer clic en campo URL para ver el detalle de la generación como se muestra a continuación:

    Método Actualizar CIP

    Se encarga de actualizar la fecha de expiración de un cip generado; si el cip se encuentra expirado al actualizarse la fecha de expiración este pasa a estado generado. Se requieren los siguientes parámetros de invocación:

    • URL: http://services.pagoefectivo.pe

    • Punto de entrada: /v1/cips/{id}

    • Método: PATCH

    Response: Ingresar el CIP y la fecha de expiración, luego clic en Actualizar par mostrar la siguiente respuesta:

    Luego se verifica el CipURL para validar la actualización de la Fecha de expiración.

    A continuación se muestra el código desarrollado en C# para invocar el Método actualizar CIP.

    //Método Actualizar CIP

...
using (var client = new HttpClient())
{
    client.BaseAddress = new Uri(responseAuthz.url);
    client.DefaultRequestHeaders.Authorization = new AuthenticationHeaderValue("Bearer", responseAuthz.Data.Token);

    var cip= new CipUpdate
    {
        DateExpiry = model.DateExpiry
    };

    string stringData = JsonConvert.SerializeObject(cip);
    var contentData = new StringContent(stringData, System.Text.Encoding.UTF8, "application/json");
    var request = new HttpRequestMessage(new HttpMethod("PATCH"), responseAuthz.url + "v1/cips/" + model.Data[0].Cip) { Content = contentData };
    HttpResponseMessage response = client.SendAsync(request).Result;

    if (response.IsSuccessStatusCode)
    {
        ViewBag.code = StatusCode((int)response.StatusCode).StatusCode;
        ViewBag.status = response.StatusCode;
        ViewBag.response = response.Content.ReadAsStringAsync().Result;
    }
    else
    {
        ViewBag.code = StatusCode((int)response.StatusCode).StatusCode;
        ViewBag.status = response.StatusCode;
        ViewBag.response = response.Content.ReadAsStringAsync().Result;
    }

}

return View();

    Método Eliminar CIP

    Se encarga de eliminar lógicamente el Número de CIP siempre en cuando este aún se encuentre en estado generado, Para consumir el método Eliminar ingresar el número de CIP. Se requieren los siguientes parámetros de invocación:

    • URL: http://services.pagoefectivo.pe

    • Punto de entrada: /v1/cips/{id}

    • Método: DELETE

    Response: Ingresar el CIP, luego clic en Eliminar y Muestra la siguiente respuesta:

    Al verificar el CipUrl no muestra ninguna información habiendo hecho la eliminación lógica del Cip como se muestra a continuación:

    A continuación se muestra el código desarrollado en C# para invocar el Método eliminar CIP.

using (var client = new HttpClient())
